documentation
This message is issued upon 1) Arrrival, departure or run-through in agreed reporting points and/or 2) Attainment of the agreed initial running time and/or 3) A new divergence between nominal and actual being achieved in excess of the agreed threshold value 4) as a response to the EnquiryTrainsAtReportingLocationMessage.  There will only be one train reported per message and will include one response per train at a location.
