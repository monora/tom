Train Object Model¶

This repository contains a proposition to extend the Combined Route Model of the JSG Train Object Model. See Route Domain Model.

We provide examples which can be tested and investigated. See

  • Example 1 Train from Amsterdam to Frankfurt over two different handovers

  • Example 2 Train with three IMs involved

  • Example 3 Train example of Working Group Annex 4

Features¶

  • TOM Jupyter Notebook (no account necessary)

  • Gitpod Github account needed. You get a VS-Code workspace with all libraries installed.
