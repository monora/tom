Example Train Annex 4Â¶

Here we investigate the routing specification for example from train-annex-4.yml

from tom.util import example
from tom.tom import make_train_from_yml, TrainRun, RouteSection
from tom.plot import plot_train

Load example annex 4 from yaml specification Notice the route sections which have a departure time. These are considered as route construction starts Only one section in a route may be a construction start.

pattern = 'annex-4.yml'
train_specs, t_spec_file = example('../tests/data', pattern)
print(t_spec_file.read_text())
